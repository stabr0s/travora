# Zasady projektowania

1.
Każda funkcja maksymalnie 3 kliknięcia.

2.
Najważniejsze informacje zawsze widoczne.

3.
Mapa i Plan są najważniejszymi ekranami.

4.
Projekt ma przypominać aplikacje Apple.

5.
Minimalizm ponad ilość funkcji.

6.
Każda nowa funkcja musi pasować do istniejącego stylu.
